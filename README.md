# FinWise – Smart Expense Tracker using ML

FinWise is a smart expense tracking web application that allows users to add daily expenses and automatically categorizes them using a machine learning model.

## Features

- Add daily expenses
- Automatic expense categorization using ML
- Dashboard with pie and line charts
- Edit and delete expense records
- Simple savings suggestions

## Tech Stack

- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Python, Flask
- ML: Scikit-learn (TF-IDF + Logistic Regression)
- Database: SQLite
- Charts: Chart.js
